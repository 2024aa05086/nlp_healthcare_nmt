# Healthcare Neural Machine Translation System

A comprehensive Neural Machine Translation system specialized for healthcare domain with automatic BLEU score evaluation.

## Features

- **Healthcare Domain Translation**: Specialized for medical texts
- **Multiple Language Support**: English → French, Spanish, German
- **BLEU Score Evaluation**: Automatic translation quality assessment
- **Detailed Analysis**: N-gram precision breakdown, brevity penalty
- **User-Friendly Interface**: Gradio web interface
- **Multiple Evaluation**: Compare multiple candidate translations
- **Translation History**: Track recent translations
